# 🎬 Mood-Based Movie Recommendation System

## 📌 Overview
This project is an MSc-level movie recommendation system that suggests movies based on user mood using TMDb API and content-based filtering.

## 🎯 Objectives
- Fetch real-time movie data using TMDb API
- Clean and preprocess movie dataset
- Map user moods to movie genres
- Recommend movies using a scoring algorithm

## 🛠️ Technologies Used
- Python
- Pandas
- Requests
- Google Colab
- HTML/CSS
- GitHub

## ⚙️ Methodology
1. Data Collection from TMDb API
2. Data Cleaning and Preprocessing
3. Genre Mapping
4. Mood-Based Filtering
5. Scoring System based on:
   - Genre match
   - Keywords
   - Ratings
   - Popularity

## 📊 Features
- Mood-based recommendations
- Clean dataset generation
- HTML styled output
- Interactive user input

## 🚀 How to Run
1. Open notebook in Google Colab
2. Add your TMDb API key
3. Run all cells
4. Enter your mood
